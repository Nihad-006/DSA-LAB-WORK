# Data Structures & Algorithms

This repository contains implementations and exercises for core data structures and algorithms, organized into **Home Work** and **Lab Work** sections.

## 📁 Home Work

| Topic | Description |
|---|---|
| [Fibonaci](./Home%20work/Fibonaci) | Fibonacci sequence implementations (recursive, iterative, memoized) |
| [HEAP](./Home%20work/HEAP) | Max Heap / Min Heap implementations (array-based), insertion, deletion, heapify, heap sort |
| [Queue](./Home%20work/Queue) | Queue data structure implementations (array/linked list based) — enqueue, dequeue operations |
| [Stack](./Home%20work/Stack) | Stack data structure implementations — push, pop, peek operations |

## 📁 Lab Work

| Topic | Description |
|---|---|
| [Dynamic array](./LAB%20WORK/Dynamic%20array) | Dynamic array implementation with resizing logic |
| [Link_list](./LAB%20WORK/Link_list) | Linked list implementations (singly, doubly, circular) |
| [Recursion](./LAB%20WORK/Recursion) | Recursive algorithm exercises and problem sets |
| [Searching](./LAB%20WORK/Searching) | Searching algorithms (linear search, binary search, etc.) |

## 🛠 Language

All code is written in **C++**.

## ▶️ How to Run

```bash
g++ filename.cpp -o output
./output
```

## 📌 Notes

- Each folder contains standalone `.cpp` files demonstrating the respective concept.
- Code is written for learning/practice purposes as part of coursework.
